Overview

Commons Monitoring (Sandbox) shares mailing lists with all the other Commons Components. To make it easier for people to only read messages related to components they are interested in, the convention in Commons is to prefix the subject line of messages with the component's name, for example:

  • [monitoring] Problem with the ...

Questions related to the usage of Commons Monitoring (Sandbox) should be posted to the User List.
The Developer List is for questions and discussion related to the development of Commons Monitoring (Sandbox).
Please do not cross-post; developers are also subscribed to the user list.

Note: please don't send patches or attachments to any of the mailing lists. Patches are best handled via the Issue Tracking system. Otherwise, please upload the file to a public server and include the URL in the mail.
